# Scope

This list tracks **Agent Skills** — modular, reusable capability packages that extend AI coding agents (Claude Code, Codex, Cursor, Gemini CLI, Copilot, Antigravity, etc.) via a `SKILL.md` (or equivalent) convention.

## In scope

- **Skill collections** — vendor or community bundles of ready-to-use skills.
- **Domain-specific skill packs** — product management, marketing, SEO, research, science, finance, android dev, frontend taste, etc.
- **Skill tooling** — loaders, registries, marketplaces, hubs, package managers, builders, creators.
- **Skill-adjacent infrastructure** — memory layers, context databases, evaluation benchmarks explicitly for skills.
- **Specifications & awesome-lists** focused on the Agent Skills format.

## Out of scope

- Generic agent harnesses / CLIs (those belong in [Local-AI-Agent-Resources](https://github.com/danielrosehill/Local-AI-Agent-Resources)).
- Pure prompt libraries with no skill packaging.
- Cloud-only SaaS agents with no installable skill artifact.

## Grey areas

- **Memory systems** are included only when they explicitly target skill persistence/reuse.
- **Frameworks** are included only when skills are a first-class primitive.
- **Single-skill repos** are included when the skill is substantial or the concept is novel.
